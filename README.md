# Lightweight IDS for Small Networks

## Overview
This project is a lightweight intrusion detection system designed for small-scale networks. It provides a simple web-based interface for viewing and analysing security alerts.


## Technologies Used
- Python
- Flask (for web-page framework)
- HTML/CSS

## Deployment
The system is deployed using Render and can be accessed using htis link: https://lightweight-ids.onrender.com/
